CORE FEATURES

为无障碍创作
重新定义交互

纯语音操控

彻底抛弃鼠标键盘，从起笔到成画，全程只需开口指令。

语义空间定位

无需报坐标，说"左上角"、"中间偏右"，AI自动映射画布位置。

视障听觉闭环

画笔落处必有回声。AI实时语音播报画面内容，视障用户心中有数。

模糊意图解析

说"画个像天空一样的圆"或"把它变大点"，大模型精准理解并执行。

TECH ARCHITECTURE

双通道引擎
响应零延迟

为了兼顾复杂语义的理解力与绘图操作的即时性，我们首创了"快慢双通道"架构。慢通道由通义千问大模型处理复杂意图，快通道由前端本地拦截撤销/清空等指令，确保体验流畅。

ASR 语音采集 & VAD 静音检测
通义千问 Function Calling 动作拆解
Canvas 对象化状态树渲染
TTS 强制语音确认闭环
