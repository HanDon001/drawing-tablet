## 一、项目概述

### 1.1 项目简介

VoiceCanvas 是一款**绝对零物理交互**（无鼠标、无键盘、无触屏）的纯语音控制绘图工具，专为肢体不便和视力障碍用户设计。

### 1.2 核心特性

| 特性 | 说明 |
|------|------|
| 🎤 纯语音控制 | 通过自然语言指令完成所有绘图操作 |
| ♿ 无障碍设计 | TTS 实时播报，视障用户友好 |
| 🤖 AI 驱动 | 理解模糊、复杂的语音指令 |
| 🎨 对象化绘图 | 支持图形创建、编辑、删除完整生命周期 |
| 💾 本地持久化 | 防抖自动保存，刷新不丢失 |
| 🔗 全链路追踪 | X-Request-ID 请求追踪 |

### 1.3 技术栈

| 层级 | 技术 | 版本 |
|------|------|------|
| 前端框架 | Vue 3 | ^3.4.15 |
| 构建工具 | Vite | ^5.0.11 |
| 状态管理 | Pinia | ^2.1.7 |
| HTTP 客户端 | Axios | ^1.6.5 |
| 后端框架 | FastAPI | ^0.136.3 |
| 日志系统 | Loguru | ^0.7.3 |
| 配置管理 | Pydantic Settings | ^2.13.1 |
| 语音识别 | Web Speech API (ASR) | 浏览器原生 |
| 语音合成 | Web Speech API (TTS) | 浏览器原生 |

```python
@app.middleware("http")
async def catch_exceptions_middleware(request: Request, call_next):
    request_id = request.headers.get("X-Request-ID", "unknown")
    try:
        response = await call_next(request)
        return response
    except Exception as e:
        logger.error(f"[{request_id}] 未捕获异常: {str(e)}")
```

```python
class ToolRegistry:
    _tools = {}
    
    @classmethod
    def register(cls, name):
        """注册工具装饰器"""
        def decorator(func):
            cls._tools[name] = tool(func)
            return func
        return decorator
    
    @classmethod
    def get_tools(cls) -> List:
        """获取所有已注册工具"""
        return list(cls._tools.values())
```

## 七、关键设计决策

### 7.1 为什么使用简单意图解析而不是 LLM？

**原因**：
1. 当前阶段需要快速验证 MVP
2. 避免 LLM API 的网络延迟（1-3秒）
3. 简化部署和配置

**未来计划**：
- Phase 4: 集成通义千问 LLM，支持更复杂的指令理解

### 7.2 为什么使用 transcript 时间戳？

**问题**：用户说相同命令时，Vue watch 不会触发

**解决方案**：
```typescript
// 添加时间戳确保唯一性
transcript.value = `${result}__${Date.now()}`
```

### 7.3 为什么使用防抖保存？

**问题**：频繁写入 localStorage 影响性能

**解决方案**：
```typescript
// 2秒防抖
watch(objects, () => {
  if (saveDebounceTimer) clearTimeout(saveDebounceTimer)
  saveDebounceTimer = setTimeout(() => {
    canvasStore.saveToLocal()
  }, 2000)
})
```

## 十、已知问题与限制

### 10.1 已知问题

1. **LLM 未集成**：当前使用简单正则解析，不支持复杂指令
2. **中文编码**：Windows 命令行中文显示乱码
3. **端口占用**：需要先杀死占用端口的进程

### 10.2 功能限制

1. 不支持自由画笔（海龟绘图）
2. 不支持复杂相对距离感知
3. 不支持多对象层级关系
